# PINF
Nouvelle version du projet
  
Procédure d'implémentation du projet :
  * créer une base de données vide
  * ajouter le contenu du fichier "pinf_v2.sql" dans celle-ci
  * déposer l'ensemble de l'archive sur le serveur local (décompréssée)
  * renommer le dossier racine en "pinfmoi" si ce n'est pas le cas
  
Important : 
  * la Base De Données créée doit se nommer "bdd_pinf" (sinon modifier le nom dans le fichier util/config.php)
  * si vous êtes sous Ubuntu (ou autre distribution Linux) , vous devrez modifier le fichier util/config.php (mot de passe), car celui-ci est configuré pour fonctionner sous Windows par défaut
